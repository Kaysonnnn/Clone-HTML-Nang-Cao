Công cụ & Framework

  • React

    3 năm kinh nghiệm phát triển ứng dụng với React, bao gồm Redux và React Hooks.

  • Vue.js

    2 năm kinh nghiệm sử dụng Vue.js cho các dự án thương mại điện tử và trang web giới thiệu sản phẩm.

  • Bootstrap & Tailwind

    Thành thạo trong việc sử dụng Bootstrap và Tailwind CSS để phát triển giao diện nhanh chóng và hiệu quả.

  • Git & GitHub

    Kỹ năng quản lý mã nguồn và phiên bản, sử dụng Git và GitHub cho cả dự án cá nhân và dự án nhóm.

  • Figma & Adobe XD

    Kinh nghiệm làm việc với các công cụ thiết kế để chuyển đổi mockups thành code HTML và CSS chính xác.
